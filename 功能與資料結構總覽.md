### ✍️ Manuscript（原稿）路徑：`/book/:bookId`（預設分頁）

三欄式寫作編輯器，核心寫作介面。

#### 左側欄 — 章節樹

| 功能 | 說明 |
|------|------|
| 章節列表 | 顯示所有章節，點擊切換載入 |
| 角色摘要 | 顯示前 3 位角色，附連結至角色頁 |

#### 中間欄 — 編輯器

| 功能 | 說明 |
|------|------|
| 章節標題編輯 | 點擊即可在頂端列修改章節標題 |
| 字數顯示 | 即時顯示本章字數 |
| 儲存狀態 | 顯示「typing / saving / saved HH:MM / error」 |
| 插入圖片 | 上傳圖片至 Drive，嵌入 Markdown |
| 專注模式 | 隱藏側欄，全螢幕寫作（F11 或 Cmd+.） |
| 富文字編輯 | 支援標題 H1–H3、粗體、斜體、清單（Tiptap） |
| 自動儲存 | 停止打字 3 秒後自動上傳至 Drive |
| 衝突偵測 | 比對雲端修訂版本 ID，若遠端已更新則提示覆寫確認 |
| 圖片擴展 | 載入時將 `drive:ASSET_ID` 轉為 Blob URL 顯示 |
| 圖片收合 | 儲存時將 Blob URL 轉回 `drive:ASSET_ID` |

#### 右側欄 — 側邊面板

| 功能 | 說明 |
|------|------|
| 筆記 | Textarea 記錄專案備忘，失焦自動儲存 |
| 待辦清單 | 新增／勾選待辦事項 |
| 今日字數進度 | 進度條顯示今日已寫字數 vs 每日目標 |
| 調整每日目標 | 點擊按鈕修改每日字數目標 |

#### 鍵盤快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| Cmd/Ctrl + S | 強制儲存 |
| F11 / Cmd/Ctrl + . | 切換專注模式 |

### 🗂️ Plot（情節）路徑：`/book/:bookId/plot`

看板式情節規劃工具，結構為：幕（Act）→ 章（Chapter）→ 場景（Scene）。

| 功能 | 說明 |
|------|------|
| 新增幕 | 建立新的故事幕，預設包含一個章節 |
| 新增章節 | 在指定幕內新增章節欄位 |
| 新增場景 | 在章節內新增場景卡片（開啟 SceneModal） |
| 編輯場景 | 點擊卡片開啟 SceneModal 修改標題、摘要、標籤 |
| 拖曳排序 | 用 dnd-kit 拖曳場景卡片，支援跨章節移動 |
| 重新命名幕/章 | 直接點擊標題編輯 |
| 標籤篩選 | 從下拉選單選擇標籤，過濾顯示場景 |
| 儲存變更 | 手動點擊儲存，將 plotBoard 寫回 project.json |
| 未儲存提示 | 有未儲存變更時顯示提示文字 |

## 二、資料結構對應功能

```
Project（專案）
├── id                    ← Drive 資料夾 ID（Shelf：開啟書本用）
├── title                 ← Shelf：顯示書名；BookLayout：頁面標題
├── coverAssetId          ← Shelf：書卡封面圖（選填）
├── manuscriptFolderId    ← Manuscript：上傳章節 .md 用
├── assetsFolderId        ← Manuscript/Characters/Research：上傳圖片用
├── projectFileId         ← 所有功能儲存時寫入的 project.json 識別碼
│
├── chapters[]            ← Manuscript：章節列表、切換、儲存字數
│   ├── id                ← 識別符（如 "ch_001"）
│   ├── title             ← Manuscript：章節標題顯示與編輯
│   ├── order             ← Manuscript：ChapterTree 排序
│   ├── fileId            ← Manuscript：Drive .md 檔案識別碼
│   ├── wordCount         ← Overview：總字數統計
│   └── rev               ← Manuscript：衝突偵測版本號
│
├── characters[]          ← Characters：角色資料庫
│   ├── id
│   ├── name              ← Characters：角色姓名；Manuscript 左側欄顯示
│   ├── label             ← Characters：分頁篩選類別
│   ├── aliases[]         ← Characters：別名列表
│   ├── description       ← Characters：角色描述
│   ├── portraitAssetId   ← Characters：肖像圖片（Drive Asset）
│   ├── tags[]            ← Characters：標籤（擴充用）
│   └── order             ← Characters：顯示排序
│
├── research[]            ← Research：研究素材庫
│   ├── id
│   ├── title             ← Research：素材標題
│   ├── description       ← Research：描述文字
│   ├── imageAssetId      ← Research：素材圖片
│   ├── tags[]            ← Research：標籤篩選
│   ├── sourceUrl         ← Research：來源網址連結
│   └── order             ← Research：排序
│
├── notes                 ← Manuscript 右欄：專案筆記 Textarea
│
├── todos[]               ← Manuscript 右欄：待辦清單
│   ├── id
│   ├── text              ← 待辦事項內容
│   └── done              ← 勾選狀態
│
├── plotBoard             ← Plot：情節看板
│   └── acts[]
│       ├── id / title    ← Plot：幕標題（可編輯）
│       └── chapters[]
│           ├── id / title ← Plot：章節欄標題
│           └── scenes[]
│               ├── id / title  ← Plot：場景卡標題
│               ├── summary     ← Plot：場景摘要
│               ├── imageAssetId ← Plot：場景圖片（選填）
│               ├── tags[]      ← Plot：標籤篩選用
│               ├── chapterRef  ← Plot：對應原稿章節 ID
│               └── order       ← Plot：拖曳排序
│
├── dailyWordGoal         ← Overview：每日字數目標；Manuscript 右欄：進度條
├── projectWordGoal       ← Overview：全書字數目標（預設 80,000）
│
├── wordHistory[]         ← Overview：7 日字數長條圖資料來源
│   ├── date              ← "YYYY-MM-DD"
│   └── total             ← 當日累積總字數
│
├── milestones[]          ← Layout：里程碑面板
│   ├── id / label        ← 里程碑名稱
│   ├── createdAt         ← 建立時間
│   └── driveRevisionId   ← 對應的 Drive 修訂版本（可回溯）
│
├── updatedAt             ← Shelf：書卡顯示最後編輯時間
└── rev                   ← 全域衝突偵測版本計數器
```

## 三、Google Drive 儲存對應

| Drive 物件 | 內容 | 用於哪個功能 |
|-----------|------|------------|
| `project.json` | 整個 Project 物件（chapters meta、characters、research、plotBoard 等） | 幾乎所有功能的讀寫 |
| `ch_001.md` ⋯ | 章節 Markdown 內文（含 `drive:ASSET_ID` 圖片引用） | Manuscript 編輯器 |
| 圖片 Asset 檔案 | 肖像、研究圖片、場景圖片、章節內嵌圖 | Characters / Research / Plot / Manuscript |
| `appDataFolder` | Drive 隱藏儲存區，使用者 Drive 介面不可見 | 所有資料的根儲存位置 |
